# GaiaEyes iOS

This repository contains the Gaia Eyes iOS application. Recent updates add the user symptom pipeline, including:

- Dashboard Symptoms tile with quick logging and 30-day sparkline visualisation.
- Offline queueing for symptom submissions with automatic retry on launch or refresh.
- Weekly trend overlays highlighting high symptom activity days.

See [`docs/ios/Symptoms_UI.md`](docs/ios/Symptoms_UI.md) for implementation notes and UI snapshots.
