# Symptoms UI Overview

The iOS dashboard now surfaces user symptom logging end to end:

- **Symptoms tile**: Displays today’s logged count, queued offline entries, and a 30-day sparkline coloured by mean severity.
- **Quick logging sheet**: Uses grouped compact chips, supports multi-select in one save action, shows suggested symptoms based on current context, and applies one shared 1–10 severity value (default `5`) plus optional notes across the selected set.
- **Offline queue**: Events are saved locally when offline and automatically flushed on launch or refresh.
- **Trend overlays**: Weekly trend charts now show soft bands for days with high symptom activity.
